Updates

Already have AlphaConcept installed? Grab the latest in-place update below — it patches your existing app in seconds, no full reinstall and no re-copying the ~90 MB application.

First time installing? You want the full installer on the download page instead — these packages only patch an app that's already installed.

0.1.10

Latest
Released 2026-07-17 · ~163 KB · Windows 10 / 11
⬇ Download update

What's new

  • Real-time network priority. Screen video and keyboard/mouse input are now tagged high priority (DSCP EF) so they stay as close to real time as possible when the connection is busy.
  • Under bandwidth pressure the host now keeps the frame rate smooth and lowers resolution instead of stuttering.
  • To enforce it on Windows, run the included Enable Real-Time Priority (Admin).cmd once on both PCs. For internet links, also prioritise DSCP 46 (or the host's IP) on your router.

What's new

  • Brand-new "warm studio" look — cream light mode and a cozy dark mode, soft pastel accents, and smoother animations.
  • Light / dark is remembered per device (toggle in the sidebar).
  • Presentation only — pairing, sessions, security, and shortcuts all work exactly as before.

How to apply an update

  1. Download and extract the .zip above.
  2. Close AlphaConcept (the updater will also close it for you).
  3. Double-click apply-update.cmd. If it can't find the app, drag the folder containing AlphaConcept.exe onto it.
  4. Start AlphaConcept again — you're on the new version.

To roll back: delete <app>\resources\app\out and rename out.bak back to out.
