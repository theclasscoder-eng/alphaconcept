Windows · Self-hostable · Peer-to-peer

Your other computer, right here.

AlphaConcept is a private remote-desktop app for Windows. Pair two of your own machines and control one from the other over an encrypted peer-to-peer connection — no accounts, no middle-man streaming your screen.

⬇ Download for Windows All releases
Windows 10 / 11 · free & open source · installer attached to the latest release
🔒

Encrypted end-to-end

Screen and input travel directly between your devices over WebRTC (DTLS). A tiny signaling server only introduces paired devices.

🤝

Explicit pairing

Devices pair with a one-time code and verify by fingerprint. You approve each session — or grant trusted devices unattended access.

🔑

Per-connection codes

Optionally require a secret code per device, proven over the encrypted channel before input is ever allowed.

🖱️

Full remote control

Real mouse and keyboard control with correct high-DPI mapping, plus an instant emergency-stop shortcut to take back control.

🎨

Light & dark

A calm, hand-tuned interface with a warm cream light mode and a cozy dark mode — remembered per device.

🏠

Self-hostable

Run your own signaling server on your LAN or a small VPS. Your devices, your infrastructure, your data.

Install

  1. Click Download for Windows and grab the AlphaConcept-Setup-*.exe attached to the latest release.
  2. Run it. Windows SmartScreen may warn that the app is unsigned — choose More info → Run anyway.
  3. Install it on both computers you want to pair.
  4. Point both apps at your signaling server in Settings → Connection, then pair them.

Already installed? Update in place

You don't need to reinstall the whole app for a new version. Each release ships a small update package (~160 KB) that patches your existing install in seconds.

  • Open the updates page and download the latest .zip.
  • Extract it and run apply-update.cmd.
  • Start AlphaConcept again — that's it.
